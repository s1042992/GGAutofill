# GGAutofill
### 使用方式有兩種

## 電腦上：
1. 在Chrome瀏覽器中 增加一個書籤(快捷鍵:ctrl + shift + o)，
2. **名稱** 隨便打
3. **網址** 貼上GGAutofill.js的code，最前面加上 **"javascript:"**
e.g.
![](https://i.imgur.com/EH1fMzO.png)
4. 在那行code中找到 input_list[1].setAttribute('value', 'XXXXXX')，**其中XXXXX改成自己工號** (''<-單引號請留著)
5. 開啟表單網址，點選此書籤即可完成~
